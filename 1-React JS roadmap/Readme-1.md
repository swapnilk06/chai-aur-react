#### Why react was created?
- Ghost/Phantom message problem
- No consistency in UI
- e.g. ham ne message padh liya hai, to bhi notification kiyu dikh raha hai

##### Problem 
- State hai jo -> `JS` manage karta hai
- State is fancy name for -> all variables
- UI manage bt -> `DOM`

##### Solution 
- In between `DOM` & JS `State` in dono main baat honi chaiye
- In dono main kabhi bhi `sync` nahi ho raha the
- That problem solve come --> `React`

- React 1st adopt by **Khan academy** after that **Unsplash**  

- React library consistency work --> Every one by impress by that

#### Don't learn React if
- you don't know how JS works OR DOM works
